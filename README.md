# 🎯 Problem Definition

Air pollution has become one of the major environmental and public health concerns in India. Increasing levels of pollutants such as PM2.5, PM10, NO₂, SO₂, CO, and O₃ negatively affect human health, climate, and quality of life.

The objective of this project is to analyze real-time Air Quality Index (AQI) data collected from multiple monitoring stations across India to:

* Identify pollution trends
* Compare air quality across cities and regions
* Analyze pollutant relationships
* Detect high-risk pollution zones
* Generate insights through statistical analysis and visualizations

Using Python-based Exploratory Data Analysis (EDA), this project aims to transform raw environmental data into meaningful insights that can support environmental monitoring and decision-making.

The dataset includes real-time AQI measurements from various monitoring stations across India and contains pollutant concentration values monitored by the Central Pollution Control Board (CPCB). ([Data.gov.in][1])
